### Markov chain simulation for rock paper and scissors

Follow these steps
1. Move to project directory `pwd` should yield `*/Lab_02/Project`
2. Turn on debugging by setting the environment variable DEBUG to "true" `export DEBUG=true`
3. Download dependencies `pip install -r requirements.txt` 
4. Run the simulation `python3 main.py`
5. Read the simulation results from the console and check the generated plot under project's directory.
