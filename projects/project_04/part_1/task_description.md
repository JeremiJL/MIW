Program 1
For given data in files daneXX.txt, adjust the template to optimise the model performance. Select at least
8 data files from the available. Note that the network needs to be adjusted for each file.

The template is available in the project directory.

For each file, find an optimal combination of
1. Activatation function and its derivative (using ReLU for all examples will not be accepted),
2. Strategy for initialising weights and biases,
3. Learning rate,
4. Regularisation parameters.

Your goal is optimising
1. R2
score (more is better, training can be stopped at 0.95)
2. Epoch count (fewer is better)
3. Number of neurons/parameters (fewer is better)

Create a report outlining your work. Figure 1 presents an example of how a report for a single data file can
look. Note the additional information about your changes in the bottom-right corner.
