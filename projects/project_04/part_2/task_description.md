Program 1
Use the keras package with the Functional API.

For given data in files daneXX.txt, design and train a regression model. Select at least 8 data files from the
available to test your network.
The architecture is left at your discretion. Minimise the size of the network and the number of epochs needed
to train it. Zero error is not required but the grade is proportional to the accuracy of the model.
Plot the training and validation losses, note the final error on the testing set.

Grading
- 2 points - Model architecture and training
- 1 point - Graphs and analysis
3 points total.
